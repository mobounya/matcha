# matcha

## Env variables

| Name          | Required | Type      | Default value | Description                            |
| ------------- | -------- | --------- | ------------- | -------------------------------------- |
| `APP_PORT`    | true     | `integer` | -             | Port that express will be listening on |
| `DB_USER`     | true     | `string`  | -             |                                        |
| `DB_PASSWORD` | true     | `string`  | -             |                                        |

## Getting Started

run the development server:

```bash
docker-compose up
```
